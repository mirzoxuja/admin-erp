# Admin ERP — IT Education Platform

An ERP-style web application for an Uzbek-language online IT education platform.

## Stack
- React + TypeScript + Vite
- Tailwind CSS

## Pages
- AddGroup.tsx
- AddStudent.tsx
- Attendence.tsx
- Dashboard.tsx
- GroupDetail.tsx
- Groups.tsx
- NotFound.tsx
- Schedule.tsx
- StudentDetail.tsx
- Students.tsx

## Run locally
npm install
npm run dev
